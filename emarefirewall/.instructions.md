```python
fw = self._detect_type(server_id)
if fw == 'emareos':
    cmd = '/emare ...'
else:
    cmd = 'linux-command ...'
```

## 📐 Kod Pattern'ları

### API Response Formatı
```json
{"success": true, "data": {...}}
{"success": false, "error": "Hata mesajı"}
```

### Naming
- Python: `snake_case`, private `_` prefix
- JavaScript: `camelCase`, prefix: `fw` (firewall), `fwNet` (network), `fwL7` (L7)
- HTML ID: `kebab-case` (panel-id), `camelCase` (input-id)
- API path: `kebab-case` (`/dns-lookup`, `/port-check`)
- Emare OS komut: `/emare` prefix

## 🧪 Mock Executor Kuralları

`app.py`'deki mock, `if "komut" in cmd:` ile substring eşleşme yapar.
- **Daha spesifik kontroller ÖNCE gelmeli** (örn: `interfaces print stats` → `interfaces print` öncesinde)
- Emare OS mock verisi `_eo_` prefix'li tuple string
- Mock data terse formatı: ` 0   key=value key=value\n`
- `_exec_multi` batched komut gönderir — SEP fallback mekanizması var
